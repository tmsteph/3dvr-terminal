3dvr-terminal

Your command-line ChatGPT assistant with clipboard magic

🛠️ Features

  • Ask ChatGPT from your terminal
  • Auto-copies code blocks to your clipboard
  • Loop mode for interactive Q&A
  • One-line install script

🚀 Install


git clone https://github.com/tmsteph/3dvr-terminal.git
cd 3dvr-terminal
chmod +x *.sh
./install.sh

Then add this to your shell config:

alias askgpt="$PWD/askgpt.sh"

💬 Example


askgpt how do I center a div in CSS?
